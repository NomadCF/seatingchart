The Project:

Classroom Seating Planner is a privacy-conscious classroom management and seating-design application that helps educators build class rosters, create accurate classroom layouts, assign students manually or automatically, evaluate seating requirements, and produce audience-appropriate print and sharing materials.
The application guides users through five primary stages: Class Setup, Room Design, Seat Students, Review, and Save & Share. It supports both traditional grid-based seating charts and detailed Freeform classroom layouts, while incorporating student needs, accessibility considerations, group relationships, seating restrictions, zones, room objects, and teacher-defined rules.
Classroom Seating Planner is designed to run locally as a self-contained application, with encrypted browser saving, backups, snapshots, recovery tools, and optional Google Drive and Google Classroom integration. It emphasizes student-data privacy by allowing teachers to control which names, details, and note categories appear in printed charts or read-only sharing packages.

Main Features

- Five-stage guided workflow
  Organizes classroom planning into Class Setup, Room Design, Seat Students, Review, and Save & Share.

- Multiple class and roster management
  Creates and maintains multiple classes, student records, archived rosters, class copies, term rollovers, and class-record comparisons or merges.

- CSV, Google Classroom, and roster reconciliation
  Imports student data from CSV files and, when configured, retrieves read-only Google Classroom rosters for review and reconciliation before changes are applied.

- Student information and categorized notes
  Stores names, IDs, grades, nicknames, requirements, attendance information, and public, substitute, or private notes.

- Student groups and classroom zones
  Defines instructional groups, behavior groupings, reserved locations, classroom zones, and relationships between students, groups, seats, and room areas.

- Detailed seating rules and requirements
  Supports together, avoid, spread, anchor, front or back placement, accessibility, zone, room-object distance, and individual student requirements.

- Standard Grid room design
  Builds structured row-and-column seating charts with seats, teacher areas, tables, doors, walls, windows, boards, projectors, walkways, carpets, ADA spaces, and custom room objects.

- Advanced Freeform room design
  Positions, resizes, rotates, aligns, groups, layers, locks, and arranges classroom objects on a canvas representing the physical room.

- Freeform layout assistance
  Includes collision handling, magnetic alignment guides, minimap navigation, object history, layout presets, print boundaries, overlap detection, and room-integrity audits.

- Manual student placement
  Allows teachers to drag, swap, move, clear, lock, or keyboard-place students while preserving intentional assignments and reserved seats.

- Automatic seating-plan generation
  Produces multiple seating candidates based on classroom rules, student needs, room geometry, existing placements, and teacher-selected priorities.

- Explainable seating recommendations
  Shows why generated seating options differ by identifying rule matches, unmet requirements, conflicts, student movement, and other decision factors.

- Preflight conflict inspection
  Detects problems such as insufficient seats, inaccessible required seating, contradictory rules, invalid locked assignments, duplicate students, and unavailable zones.

- Seating review and analysis
  Identifies unassigned students, rule violations, group outcomes, inaccessible placements, overlapping objects, hidden items, and damaged or inconsistent data.

- Today Mode and temporary classroom changes
  Supports attendance, absent students, temporary guests, daily notes, and seating only the students currently present.

- Named seating plans and layout history
  Saves, compares, restores, and manages different room arrangements or student-assignment versions.

- Undo, redo, snapshots, and selective restore
  Protects work through editing history, automatic and manual snapshots, restore points, and restoration of selected classes or data sections.

- Encrypted saving and recovery
  Supports encrypted browser autosave, encrypted downloaded backups, linked working files, password rotation, startup recovery, and optional separately stored recovery packages.

- Save-health and conflict protection
  Monitors storage status, maintains backup records, guards against interrupted writes, detects multi-tab editing, and provides conflict-resolution options.

- Optional Google Drive storage
  Saves and loads encrypted planner files through Google Drive, with file management, revisions, reconnect status, duplicate-file handling, and revision-aware conflict detection.

- Privacy-controlled printing
  Produces print-as-seen, names-only, substitute, or detailed charts with selectable student fields, notes, groups, zones, room information, paper sizes, scaling, tiling, and crop marks.

- Purpose-limited read-only sharing packages
  Creates teacher, substitute, student-facing, support-team, anonymous, or room-only HTML packages containing only the information selected for that audience.

- Workspace security controls
  Includes encrypted data, Settings protection, page locking, automatic locking, visibility or Eye Mode, deliberate sensitive-note revealing, and secure data-reset workflows.

- Responsive and accessible interface
  Supports desktop, tablet, and mobile workflows, touch placement, keyboard navigation, focus-managed dialogs, screen-reader announcements, reduced motion, collapsible panels, and focus mode.

- Integrated guidance and documentation
  Provides Quick Start instructions, searchable reference help, guided lessons, practice classes, contextual “Guide Me” tools, saved learning progress, diagnostics, and troubleshooting information.

- Customizable appearance
  Includes multiple accessible themes, display preferences, collapsible workspaces, configurable defaults, privacy views, and classroom-presentation modes.

- Portable and hosted deployment options
  Can run as a self-contained HTML application or as a hosted, installable Progressive Web App with offline support and deployment verification.

- Privacy-focused local operation
  Keeps classroom information in the browser unless the user deliberately saves, exports, prints, or connects an approved external service.
